COMMENT MANIPULER MONGODB ? 
------------------
1. Aller sur : https://cloud.mongodb.com/
2. Créons un compte
3. Choisissons l'hébergement gratuit
4. Créeer un accès DB sur son compte
5. Data service -> Network access : ajouter son ip ou TOUTES les IP.
6. Télécharger MONGODB COMPASS pour voir la base de donnée distante sur une meilleure interface que le web
7. Sur cloud.mongodb : Deployment > Database > Connect 
8. Nous sélectionnons une connexion avec Compass
9. Nous induqons que nous avons déjà une base de données
10. Il vous dira de taper le lien suivant :
mongodb+srv://<username>:<password>@cluster0.ebgz2ij.mongodb.net/
* <username> : nom d'utilisateur
* <password> : mot de passe
11. Nous sommes dans notre base de données
